I like sass. It lets you do cool stuff like scaling text or scaling containers very easily. This page should demonstrate it well enough. Below is an an example of how the text can be done: 

    
.flow-text {
  $i: 0;

  @while $i <= 20 {
 @media only screen and (min-width: 360 + $i * 100px) {
   font-size: 1rem * (1 + 0.02 * $i);
 }

 $i: $i + 1;
  }

  @media only screen and (max-width: 360px) {
 font-size: 1rem;
  }
}
